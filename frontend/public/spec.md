# AI 考试系统 - 系统规格说明书

## 1. 系统概览
**AI 考试系统** 是一个专为教育环境设计的轻量级在线考试平台。它为教师提供了创建和管理考试的完整工作流，同时也为学生提供了参加考试和查看成绩的便捷途径。该系统采用 **React** 前端和 **Supabase** 后端构建，注重安全性与易用性。

## 2. 技术架构

### 2.1 技术栈
- **前端**: React (Vite), 原生 CSS, React Router
- **后端**: Supabase (PostgreSQL, Auth, Edge Functions)
- **认证**: Supabase Auth (邮箱/密码)
- **数据库**: PostgreSQL (启用 RLS 安全策略)

### 2.2 安全机制
- **数据隔离**: 使用行级安全策略 (RLS) 确保用户只能访问授权数据。
- **服务端评分**: 考试评分逻辑在数据库层执行，防止客户端作弊。
- **安全认证**: 完善的注册、登录及密码管理流程。

## 3. 核心功能

### 3.1 教师端
- **仪表盘**: 查看考试概况和题库状态。
- **题库管理**:
    - 手动创建题目（支持 2-5 个动态选项）。
    - 批量导入题目（支持 CSV/JSON 格式）。
- **考试管理**: 创建考试、设置时长与总分。
- **成绩分析**: 查看学生考试成绩与排名。

### 3.2 学生端
- **考试中心**: 查看可用考试列表。
- **在线考试**:
    - 沉浸式答题界面。
    - 实时倒计时。
    - 答题进度导航。
- **成绩回顾**: 考后即时查看分数与错题解析。

## 4. 数据库设计
- **profiles**: 用户档案 (角色、头像)。
- **questions**: 题库 (内容、选项、答案)。
- **exams**: 考试配置 (题目集合、时长)。
- **exam_results**: 考试记录 (分数、答题详情)。
